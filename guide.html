HardMon User Guide

1. Introduction

HardMon is a hardware monitoring tool that provides real-time information about your system components, including:

  • CPU (Clock Speed, Load, Temperature, Power)
  • GPU (Load, Temperature, Clock Speed, Memory Usage)
  • RAM (Usage, Load, Available Memory)
  • HDD/SSD (Temperature, Usage, Throughput)

2. Getting Started

2.1 Launching HardMon

  • Double-click the HardMon executable to start the application.
  • The main window will display hardware information in a structured view.

2.2 Main Interface Overview

The main window consists of:

  • Monitor View Panel: Displays all hardware components and their metrics.
  • Status Bar: Shows tips and shortcuts (updates every 5 seconds).
  • Menu Bar: Contains options for settings, saving data, and exiting.

3. Using HardMon

3.1 Viewing Real-Time Data

  • Hardware metrics update every second automatically.

3.2 Saving Monitoring Data

  • Click File → Save Monitoring Data.
  • Choose a location and filename (default: HardMon_YYYY-MM-DD-hhmmss.txt).
  • Click Save to export all current hardware data as a text file.

3.3 Changing Themes

HardMon supports Light and Dark themes:

  • Dark Theme: View → Dark (Default)
  • Light Theme: View → Light

Shortcuts:

  • Dark Theme: Ctrl + D
  • Light Theme: Ctrl + L

3.4 Adjusting Font Size

You can change the text size for better readability:

  • Default: View → Default Size (9pt)
  • Medium: View → Medium (11pt)
  • Large: View → Large (12pt)

Shortcuts:

  • Default Size: Ctrl + Shift + D
  • Medium Size: Ctrl + Shift + M
  • Large Size: Ctrl + Shift + L

4. Understanding Hardware Metrics

4.1 CPU Information

  • Clock: Current frequency of CPU cores.
  • Load: CPU usage per core (%).
  • Temperature: Core temperatures (°C).
  • Power: Power consumption (Watts).

4.2 GPU Information

  • Load: GPU utilization (%).
  • Temperature: GPU temperature (°C).
  • Clock: Current GPU core/memory clock speeds (MHz).
  • Memory: VRAM usage (Used/Total).

4.3 RAM Information

  • Load: Memory usage (%).
  • Usage: Breakdown of used/available/total RAM.

4.4 HDD/SSD Information

  • Temperature: Drive temperature (°C).
  • Usage: Disk activity (%).
  • Throughput: Read/Write speeds (MB/s). (if Supported)

5. Troubleshooting

5.1 Missing Data

  • The hardware does not support monitoring.
  • HardMon are not installed correctly.

5.2 High CPU Usage

  • HardMon runs a background thread for real-time updates.
  • If performance is impacted, close unnecessary applications.

5.3 Application Freezes

  • Ensure your system meets minimum requirements.
  • Restart HardMon if unresponsive.
