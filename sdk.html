HardEngine.dll is a lightweight yet powerful .NET library designed for hardware monitoring on legacy and modern Windows systems. It provides real-time access to critical system metrics, including CPU, GPU, RAM, storage, network, battery, and audio device information.

Key Features

  • Cross-Platform Compatibility – Supports .NET Framework 2.0+ and Windows Vista+
  • Low System Impact – Optimized for performance on older hardware
  • Comprehensive Monitoring – Covers CPU, GPU, RAM, Storage, Network, Battery, and Audio
  • Simple Integration – Easy-to-use API for C# and VB.NET
  • No External Dependencies – Works without requiring additional SDKs

API Reference

Core Class : HardwareMonitor

Constructor

public HardwareMonitor(bool enableBasicMode = false)

  • enableBasicMode (Optional): Reduces permissions needed (limited metrics)

Key Methods

Method Return Type Description
GetCPUInfo() CPUInfo Retrieves CPU details (temp, load, clock)
GetGPUInfo() GPUInfo Gets GPU stats (temp, usage, memory)
GetRAMInfo() RAMInfo Provides RAM usage (total, used, available)
GetStorageInfo() List<StorageInfo> Lists all drives (capacity, health, temp)
GetNetworkInfo() List<NetworkInfo> Network adapters (speed, IP, usage)
GetBatteryInfo() BatteryInfo Battery status (%, health, time remaining)
GetAudioDevices() List<AudioDevice> Audio endpoints (volume, mute status)
StartMonitoring() void Begins real-time updates
StopMonitoring() void Safely stops monitoring

Usage Example

Example 1 : Basic CPU & RAM monitoring (C#)

using System;
using HardEngine;

class Program {
    static void Main() {
        var monitor = new HardwareMonitor();
        
        var cpu = monitor.GetCPUInfo();
        var ram = monitor.GetRAMInfo();

        Console.WriteLine($"CPU: {cpu.Name}");
        Console.WriteLine($"Temperature: {cpu.Temperature}°C");
        Console.WriteLine($"Load: {cpu.Load}%");
        
        Console.WriteLine($"RAM Usage: {ram.UsedGB} / {ram.TotalGB} GB");
        
        monitor.StopMonitoring();
    }
}

Example 2 : Network & Battery Check (VB.NET)

Imports HardEngine

Module Module1
    Sub Main()
        Dim monitor As New HardwareMonitor()
        
        Dim netInfo = monitor.GetNetworkInfo()
        Dim battery = monitor.GetBatteryInfo()
        
        Console.WriteLine("Network Adapters:")
        For Each adapter In netInfo
            Console.WriteLine($"{adapter.Name}: {adapter.DownloadSpeed} Kbps")
        Next
        
        Console.WriteLine($"Battery: {battery.Percentage}% ({battery.Status})")
        
        monitor.StopMonitoring()
    End Sub
End Module

Best Practices

Performance Optimization

  • Avoid frequent polling (use StartMonitoring() for real-time updates).
  • Cache static data (e.g, CPU Name, GPU Model).

Error Handling

try {
    var gpu = monitor.GetGPUInfo();
} catch (HardwareException ex) {
    Console.WriteLine($"Error: {ex.Message}");
}

Admin VS. User-Mode

  • Admin Privileges required for:
    • CPU/GPU Temperatures.
    • Detailed storage health.
    • Battery wear level.
  • User-Mode provides basic metrics (CPU loads, RAM usage).

Troubleshooting

Issue Solution
"Access Denied" Error Run as Administrator
Missing GPU Data Install latest GPU drivers
Battery not detected Check power management settings
No network stats Enable network adapter in OS
